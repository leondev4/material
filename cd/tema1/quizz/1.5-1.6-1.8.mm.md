# Preguntas de Kahoot: Funciones Algebraicas, Trascendentes y Operaciones

## Pregunta 1: Dominio de función polinomial
- 1.5-1.6 Enunciado: ¿Cuál es el dominio de la función $ f(x) = 3x^3 - 2x^2 + 5x - 1 $?
- a) $ (-\infty, 0) \cup (0, \infty) $
- b) $ (-\infty, \infty) $
- c) $ [0, \infty) $
- d) $ (-\infty, 5] $
- Solución: b) $ (-\infty, \infty) $. Las funciones polinomiales están definidas para todos los números reales.

## Pregunta 2: Evaluación de función racional
- 1.5-1.6 Enunciado: Si $ f(x) = \frac{x^2 - 4}{x - 2} $, ¿cuál es el valor de $ f(3) $?
- a) 1
- b) 5
- c) Indefinido
- d) 0
- Solución: b) 5. Sustituyendo: $ f(3) = \frac{3^2 - 4}{3 - 2} = \frac{9 - 4}{1} = \frac{5}{1} = 5 $.

## Pregunta 3: Dominio de función racional
- 1.5-1.6 Enunciado: ¿Cuál es el dominio de $ g(x) = \frac{1}{x + 3} $?
- a) Todos los reales excepto $ x = -3 $
- b) Todos los reales excepto $ x = 3 $
- c) $ (-\infty, -3) $
- d) $ [0, \infty) $
- Solución: a) Todos los reales excepto $ x = -3 $. El denominador no puede ser cero: $ x + 3 \neq 0 \Rightarrow x \neq -3 $.

## Pregunta 4: Evaluación de función exponencial
- 1.5-1.6 Enunciado: Si $ h(x) = 2^x $, ¿cuál es el valor de $ h(4) $?
- a) 8
- b) 16
- c) 32
- d) 64
- Solución: b) 16. $ h(4) = 2^4 = 16 $.

## Pregunta 5: Dominio de función logarítmica
- 1.5-1.6 Enunciado: ¿Cuál es el dominio de $ f(x) = \ln(x - 1) $?
- a) $ (-\infty, 1) $
- b) $ (1, \infty) $
- c) $ [1, \infty) $
- d) $ (0, \infty) $
- Solución: b) $ (1, \infty) $. El argumento del logaritmo debe ser positivo: $ x - 1 > 0 \Rightarrow x > 1 $.

## Pregunta 6: Operación con funciones (suma)
- 1.5-1.6 Enunciado: Si $ f(x) = x^2 + 1 $ y $ g(x) = 2x - 3 $, ¿cuál es $ (f + g)(x) $?
- a) $ x^2 + 2x - 2 $
- b) $ x^2 + 2x + 4 $
- c) $ 2x^3 - 3x^2 + 2x - 3 $
- d) $ x^2 - 2x + 4 $
- Solución: a) $ x^2 + 2x - 2 $. $ (f + g)(x) = (x^2 + 1) + (2x - 3) = x^2 + 2x - 2 $.

## Pregunta 7: Composición de funciones
- 1.5-1.6 Enunciado: Si $ f(x) = x + 2 $ y $ g(x) = 3x $, ¿cuál es $ (f \circ g)(x) $?
- a) $ 3x + 2 $
- b) $ 3x + 6 $
- c) $ x + 6 $
- d) $ 4x + 2 $
- Solución: b) $ 3x + 6 $. $ (f \circ g)(x) = f(g(x)) = f(3x) = 3x + 2 $.

## Pregunta 8: Evaluación de función trigonométrica
- 1.5-1.6 Enunciado: ¿Cuál es el valor de $ \cos(60\degree) $?
- a) 0
- b) 1
- c) -1
- d) $\frac{1}{2}$
- Solución: d) 1/2
