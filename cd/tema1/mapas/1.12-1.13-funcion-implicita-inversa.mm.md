# 1.12 La Función Inversa
## Definición
- Función $f$ es **biyectiva** (inyectiva y sobreyectiva)
- Existe $f^{-1}$ tal que:
    - $f^{-1}(f(x)) = x$ para todo $x$ en dominio de $f$
    - $f(f^{-1}(y)) = y$ para todo $y$ en dominio de $f^{-1}$
- Gráficamente: simetría respecto a recta $y=x$

## Propiedades
- Dominio($f^{-1}$) = Rango($f$)
- Rango($f^{-1}$) = Dominio($f$)
- $(f^{-1})^{-1} = f$
- $(f \circ g)^{-1} = g^{-1} \circ f^{-1}$

## Métodos para hallar la inversa
### 1. Algebraico
- Ejemplo: $f(x) = 2x + 3$
    1. $y = 2x + 3$
    2. Intercambiar: $x = 2y + 3$
    3. Despejar $y$: $y = \frac{x-3}{2}$
    4. $f^{-1}(x) = \frac{x-3}{2}$

### 2. Gráfico
- Reflejar respecto a $y=x$

## Ejemplo 1: Función lineal
- $f(x) = \frac{1}{2}x - 4$
- Inversa: $f^{-1}(x) = 2x + 8$
- Comprobación: $f^{-1}(f(2)) = f^{-1}(-3) = 2$

## Ejemplo 2: Función racional
- $f(x) = \frac{x+1}{x-2}, \quad x \neq 2$
- Inversa: $f^{-1}(x) = \frac{2x+1}{x-1}, \quad x \neq 1$

## Ejemplo 3: Función raíz
- $f(x) = \sqrt{x-3}, \quad x \geq 3$
- Inversa: $f^{-1}(x) = x^2 + 3, \quad x \geq 0$

## Ejercicios
### Ejercicio 1
- Encontrar inversa de $f(x) = \frac{3x-1}{x+2}$

### Ejercicio 2
- Si $f(x) = x^2 - 4x + 5$ para $x \geq 2$, hallar $f^{-1}(x)$

### Ejercicio 3
- Verificar si $g(x) = \sqrt[3]{x+1}$ y $h(x) = x^3 - 1$ son inversas
