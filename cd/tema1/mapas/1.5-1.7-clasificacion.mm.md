# Clasificación de Funciones

## 1. Funciones Algebraicas
- **Definición:** Expresiones construidas con operaciones algebraicas (+, -, ×, ÷, potencias, raíces)
- **Variables relacionadas mediante polinomios**

### 1.1 Funciones Polinomiales
- **Forma:** $f(x) = a_nx^n + a_{n-1}x^{n-1} + \cdots + a_1x + a_0$
- - **Dominio:** Todos los reales
- **Ejemplos:**
  - Lineal: $f(x) = 2x - 3$
  - Cuadrática: $f(x) = x^2 - 4x + 4$
  - Cúbica: $f(x) = x^3 + 2x^2 - x + 5$
- **Ejercicios:**
  1. Dada $f(x) = 3x^2 - 5x + 1$, calcular $f(2)$
  2. Encontrar las raíces de $g(x) = x^3 - 6x^2 + 11x - 6$
  3. Graficar $h(x) = -x^2 + 4$

### 1.2 Funciones Racionales
- **Forma:** $f(x) = \frac{P(x)}{Q(x)}$ donde $P(x)$ y $Q(x)$ son polinomios
- **Dominio:** Todos los reales excepto donde $Q(x) = 0$
- **Ejemplos:**
  - $f(x) = \frac{2x+1}{x-3}$
  - $g(x) = \frac{x^2-4}{x^2+x-6}$
  - $h(x) = \frac{1}{x}$
- **Ejercicios:**
  1. Hallar dominio de $f(x) = \frac{x+2}{x^2-9}$
  2. Simplificar $g(x) = \frac{x^2-5x+6}{x^2-4}$
  3. Encontrar asíntotas de $h(x) = \frac{2x^2+1}{x^2-4}$

## 2. Funciones Trascendentes
- **Definición:** No pueden expresarse con operaciones algebraicas finitas

### 2.1 Funciones Trigonométricas
- **Basadas en relaciones angulares**
- **Ejemplos:**
  - Seno: $f(x) = \sin(x)$
  - Coseno: $f(x) = \cos(x)$
  - Tangente: $f(x) = \tan(x) = \frac{\sin(x)}{\cos(x)}$
- **Propiedades:** Periódicas, dominio y rango específicos
- **Ejercicios:**
  1. Resolver $\sin(x) = \frac{1}{2}$ en $[0, 2\pi]$
  2. Graficar $f(x) = 2\cos(3x) + 1$
  3. Verificar identidad: $\sin^2(x) + \cos^2(x) = 1$

### 2.2 Funciones Logarítmicas
- **Forma:** $f(x) = \log_a(x)$ donde $a > 0$, $a \neq 1$
- **Definición:** Inversa de la función exponencial
- **Ejemplos:**
  - Natural: $f(x) = \ln(x)$ (base $e$)
  - Decimal: $f(x) = \log_{10}(x)$
  - Base 2: $f(x) = \log_2(x)$
- **Propiedades:**
  - $\log_a(xy) = \log_a(x) + \log_a(y)$
  - $\log_a\left(\frac{x}{y}\right) = \log_a(x) - \log_a(y)$
- **Ejercicios:**
  1. Calcular $\log_2(32)$
  2. Resolver $\ln(x+1) = 2$
  3. Simplificar $\log_3(27x^2)$

### 2.3 Funciones Exponenciales
- **Forma:** $f(x) = a^x$ donde $a > 0$, $a \neq 1$
- **Propiedad clave:** La variable está en el exponente
- **Ejemplos:**
  - $f(x) = 2^x$
  - $f(x) = e^x$ (función exponencial natural)
  - $f(x) = \left(\frac{1}{2}\right)^x$
- **Ejercicios:**
  1. Resolver $3^{2x} = 27$
  2. Graficar $f(x) = e^{-x}$
  3. Comparar crecimiento de $2^x$ y $x^2$

## 3. Funciones Definidas por Partes
- **Definición:** Diferentes expresiones para diferentes intervalos del dominio
- **Forma:** $f(x) = \begin{cases} f_1(x) & \text{si } x \in D_1 \\ f_2(x) & \text{si } x \in D_2 \\ \vdots & \vdots \end{cases}$
- **Ejemplos:**
  - Valor absoluto: $f(x) = |x| = \begin{cases} x & \text{si } x \geq 0 \\ -x & \text{si } x < 0 \end{cases}$
  - Función escalón: $f(x) = \begin{cases} 0 & \text{si } x < 0 \\ 1 & \text{si } x \geq 0 \end{cases}$
  - Función por intervalos: $f(x) = \begin{cases} x^2 & \text{si } x \leq 1 \\ 2x-1 & \text{si } x > 1 \end{cases}$
- **Ejercicios:**
  1. Evaluar $f(2)$ para $f(x) = \begin{cases} 3x+1 & \text{si } x < 2 \\ x^2-1 & \text{si } x \geq 2 \end{cases}$
  2. Graficar $f(x) = \begin{cases} -x & \text{si } x < 0 \\ x^2 & \text{si } 0 \leq x \leq 2 \\ 4 & \text{si } x > 2 \end{cases}$
  3. Determinar continuidad en $x=1$ para $f(x) = \begin{cases} \frac{x^2-1}{x-1} & \text{si } x \neq 1 \\ 2 & \text{si } x = 1 \end{cases}$

## Ejercicios Integradores
1. Clasificar cada función:
   - $f(x) = \sqrt{x^3 + 2x}$ 
      - (Algebraica: irracional)
   - $g(x) = 3e^{2x} + \ln(x)$ 
      - (Trascendente)
   - $h(x) = \begin{cases} x^2 & x<0 \\ \cos(x) & x\geq0 \end{cases}$ 
      - (Definida por partes)

2. Hallar dominio y rango de:
   - $f(x) = \frac{1}{\sqrt{x-2}}$
   - $g(x) = \log_5(3x-6)$
   - $h(x) = |x-3| + 2$

3. Problema aplicado: Modelar temperatura diaria como función por partes con variación senoidal durante el día y constante durante la noche.
