# Definiciones básicas de funciones matemáticas

## Variable
### Variable independiente
- Símbolo que representa valores que pueden cambiar libremente
- Se denota comúnmente como **x**
- **Ejemplo**: En la relación tiempo-distancia: el **tiempo** es independiente

### Variable dependiente
- Símbolo cuyo valor depende de la variable independiente
- Se denota comúnmente como **y** o **f(x)**
- **Ejemplo**: En la relación tiempo-distancia: la **distancia recorrida** depende del tiempo
  $ \text{distancia} = f(\text{tiempo}) $

## Relación
### Definición
- Cualquier conjunto de pares ordenados **$(x, y)$**
- No necesariamente cada valor de x se relaciona con un único valor de y
- **Ejemplo**: $ \{(1,2), (1,3), (2,4)\} $ es una relación donde x=1 tiene dos valores de y

## Función
### Definición formal
- Una relación donde **cada valor de x tiene exactamente un valor de y**
- Notación: $ f: A \rightarrow B $ o $ y = f(x) $
- **Ejemplo 1**: $ f(x) = 2x + 3 $
- **Ejemplo 2**: $ g(x) = x^2 $
- **Contraejemplo**: $ x^2 + y^2 = 1 $ (círculo) NO es función ($x=0$ da $y=±1$)

### Prueba de la recta vertical
- Si una recta vertical corta la gráfica en **más de un punto**, NO es función
- **Ejemplo función**: Parábola $ y = x^2 $
- **Ejemplo no función**: Círculo $ x^2 + y^2 = 4 $

## Dominio
### Definición
- Conjunto de **todos los valores posibles de x** (variable independiente)
- Valores para los cuales la función está definida

### Tipos comunes de restricciones
1. **División por cero**: $ f(x) = \frac{1}{x-2} $ → Dominio: $ x \neq 2 $
2. **Raíces pares**: $ f(x) = \sqrt{x-3} $ → Dominio: $ x \geq 3 $
3. **Logaritmos**: $ f(x) = \ln(x+4) $ → Dominio: $ x > -4 $

### Ejemplos de dominio
-  $ f(x) = 3x^2 + 2 $ →  Dominio: $ \mathbb{R} $ (todos los reales)
- $ g(x) = \frac{2}{x^2-4}  $ → Dominio: $ x \neq \pm 2 $
- $ h(x) = \sqrt{9-x^2} $ → Dominio: $ -3 \leq x \leq 3 $

## Rango
### Definición
- Conjunto de **todos los valores posibles de $y$** (variable dependiente)
- Valores que la función puede producir como resultado

### Ejemplos de rango
- $ f(x) = x^2 $ → Rango: $ y \geq 0 $
- $ g(x) = \sin(x) $ → Rango: $ -1 \leq y \leq 1 $
- $ h(x) = \frac{1}{x} $ → Rango: $ y \neq 0 $

## Ejercicios resueltos
### Ejercicio 1: Identificar variable dependiente e independiente
- **Enunciado**: En la relación entre temperatura y ventas de helados
- **Solución**: 
  - Variable independiente: temperatura
  - Variable dependiente: ventas de helados
  - Notación: ventas = f(temperatura)

### Ejercicio 2: Determinar si es función
- **Enunciado**: $ \{(1,3), (2,5), (1,4), (3,6)\} $
- **Solución**: NO es función porque $x=1$ tiene dos valores de $y$ ($3$ y $4$)

### Ejercicio 3: Hallar dominio
- **Enunciado**: $ f(x) = \frac{\sqrt{x-1}}{x-5} $
- **Solución**:
  1. Raíz cuadrada: $ x-1 \geq 0 $ → $ x \geq 1 $
  2. Denominador: $ x-5 \neq 0 $ → $ x \neq 5 $
  3. Dominio: $ [1,5) \cup (5, \infty) $

## Recursos visuales
### Diagrama de función
- Dominio (A) → Función f → Rango (B)
  x → f → f(x)
  1 → x² → 1
  2 → x² → 4
  3 → x² → 9

### Comparación relación vs función
- **Relación**: Un $x$ puede tener múltiples $y$
- **Función**: Cada $x$ tiene exactamente un $y$
