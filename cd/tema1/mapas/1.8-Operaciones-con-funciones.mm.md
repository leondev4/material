# Operaciones con Funciones

## Adición (Suma)
- **Definición:**  
  $(f + g)(x) = f(x) + g(x)$
- **Dominio:**  
  $Dom(f + g) = Dom(f) ∩ Dom(g)$
- **Ejemplo:**  
  $f(x) = x² + 1$, $g(x) = 3x - 2$  
  $(f + g)(x) = (x² + 1) + (3x - 2) = x² + 3x - 1$
- **Ejercicios:**
  1. $f(x) = \sqrt{x}$, $g(x) = x + 4$. Hallar $(f + g)(x)$ y su dominio.
  2. $f(x) = 1/(x-2)$, $g(x) = 2x$. Hallar $(f + g)(3)$.

## Sustracción (Resta)
- **Definición:**  
  $(f - g)(x) = f(x) - g(x)$
- **Dominio:**  
  $Dom(f - g) = Dom(f) ∩ Dom(g)$
- **Ejemplo:**  
  $f(x) = 5x$, $g(x) = x² + 2x$  
  $(f - g)(x) = 5x - (x² + 2x) = -x² + 3x$
- **Ejercicios:**
  1. $f(x) = |x|$, $g(x) = 3x - 1$. Hallar $(f - g)(-2)$.
  2. $f(x) = \sqrt{x+1}$, $g(x) = 4 - x$. Hallar $(f - g)(x)$ y dominio.

## Multiplicación (Producto)
- **Definición:**  
  $(f · g)(x) = f(x) · g(x)$
- **Dominio:**  
  $Dom(f · g) = Dom(f) ∩ Dom(g)$
- **Ejemplo:**  
  $f(x) = x - 1$, $g(x) = x + 5$  
  $(f · g)(x) = (x - 1)(x + 5) = x² + 4x - 5$
- **Ejercicios:**
  1. $f(x) = \sqrt{x}$, $g(x) = x - 3$. Hallar $(f · g)(4)$.
  2. $f(x) = 2x + 1$, $g(x) = x²$. Hallar $(f · g)(x)$.

## División (Cociente)
- **Definición:**  
  $ \left(\frac{f} {g} \right)(x) =\frac {f(x)}  {g(x)}$, $g(x) ≠ 0$
- **Dominio:**  
  $Dom \left( \frac{f} {g} \right) = { x ∈ Dom(f) ∩ Dom(g) | g(x) ≠ 0 }$
- **Ejemplo:**  
  $f(x) = x² - 9$, $g(x) = x + 3$  
  $\left( \frac{f} {g}\right)(x) = \frac{x² - 9}{x + 3} = x - 3$, $x ≠ -3$
- **Ejercicios:**
  1. $f(x) = x² - 4$, $g(x) = x - 2$. Simplificar $(f / g)(x)$ y dar dominio.
  2. $f(x) = 1$, $g(x) = x² + 1$. Hallar $(f / g)(x)$ y su dominio.

## Composición
- **Definición:**  
  $(f ∘ g)(x) = f(g(x))$  
  (Se aplica primero $g$, luego $f$ al resultado)
- **Dominio:**  
  $Dom(f ∘ g) = { x ∈ Dom(g) | g(x) ∈ Dom(f) }$
- **Ejemplo:**  
  $f(x) = x²$, $g(x) = x + 2$  
  $(f ∘ g)(x) = f(x+2) = (x+2)² = x² + 4x + 4$  
  $(g ∘ f)(x) = g(x²) = x² + 2$ (¡No es conmutativa!)
- **Ejercicios:**
  1. $f(x) = \sqrt{x}$, $g(x) = 2x - 6$. Hallar $(f ∘ g)(x)$ y su dominio.
  2. $f(x) = 1/x$, $g(x) = x - 4$. Hallar $(g ∘ f)(x)$ y dominio.
  3. Si $f(x) = x + 1$, $g(x) = x²$, hallar $(f ∘ g)(2)$ y $(g ∘ f)(2)$.
- **Ejercicios Combinados:**
  1. Dadas $f(x)=x²-1$, $g(x)=2x$, hallar:  
     $a) (f+g)(x)$  
     $b) (f·g)(x)$  
     $c) (f∘g)(x)$  
     $d) (g∘f)(x)$
  2. Si $f(x)=\sqrt{x+2}$, $g(x)=1/x$, determinar dominio de:  
     $(f+g)(x)$, $(f/g)(x)$, $(f∘g)(x)$
