# 1.2 Intervalos en los reales y su representación gráfica

## Definición
- Subconjunto de números reales
- Se definen mediante desigualdades
- Se representan sobre la recta numérica

## Tipos de intervalos

### Intervalo abierto
- **Definición**: No incluye los extremos
- **Notación**: $(a, b)$ o $[a, b]$
- **Desigualdad**: $a < x < b$
- **Gráfica**: ○-----○
  - $(a,b)$
- **Ejemplo**: $(-2, 3)$ → $-2 < x < 3$

### Intervalo cerrado
- **Definición**: Incluye los extremos
- **Notación**: $[a, b]$
- **Desigualdad**: $a \leq x \leq b$
- **Gráfica**: ●-----●
  - $[a,b]$

- **Ejemplo**: $[1, 4]$ → $1 \leq x \leq 4$

### Intervalo semiabierto (o semicerrado)
- **Semiabierto por la derecha**: $[a, b)$
- $a \leq x < b$
- **Gráfica**: ●-----○
- **Semiabierto por la izquierda**: $(a, b]$
- $a < x \leq b$
- **Gráfica**: ○-----●

### Intervalos infinitos
- **Hacia +∞**: $(a, +\infty)$ o $[a, +\infty)$
- **Ejemplo**: $[2, +\infty)$ → $x \geq 2$
- **Hacia -∞**: $(-\infty, b)$ o $(-\infty, b]$
- **Ejemplo**: $(-\infty, -1)$ → $x < -1$
- **Todos los reales**: $(-\infty, +\infty)$

## Operaciones con intervalos

### Unión de intervalos
- **Símbolo**: $\cup$
- **Definición**: $A \cup B = \{x \mid x \in A \text{ o } x \in B\}$
- **Ejemplo**: $(-1, 2] \cup [1, 3) = (-1, 3)$

### Intersección de intervalos
- **Símbolo**: $\cap$
- **Definición**: $A \cap B = \{x \mid x \in A \text{ y } x \in B\}$
- **Ejemplo**: $[0, 4) \cap (2, 5] = (2, 4)$

## Ejercicios resueltos

### Ejercicio 1: Representar gráficamente
- $[-3, 1)$
  - ●-----○

### Ejercicio 2: Expresar como desigualdad
- $( -\infty, 5 ]$
- Solución: $x \leq 5$

### Ejercicio 3: Convertir desigualdad a intervalo
- $-2 < x \leq 3$
- Solución: $(-2, 3]$

## Ejercicios propuestos

### Nivel básico
1. Representar gráficamente: $(0, 4]$
2. Escribir en notación de intervalo: $x \geq -1$
3. Expresar como desigualdad: $(-3, 7)$

### Nivel intermedio
4. Hallar $A \cap B$ si $A = [-2, 3]$ y $B = (1, 5)$
5. Hallar $A \cup B$ si $A = (-\infty, 2]$ y $B = [0, 4)$
6. Representar en recta: $[1, 3) \cup (2, 4]$

### Nivel avanzado
7. Resolver: $[-1, 4] \cap ([0, 3) \cup (2, 5))$
8. Expresar como intervalo: $\{x \in \mathbb{R} \mid |x| < 3\}$
9. Graficar: $(-\infty, -2) \cup [0, 1] \cup (3, +\infty)$
