# Ecuación del Plano

## 1. Definición y formas de ecuación
- Plano: superficie bidimensional infinita en ℝ³
- Determinado por:
  - Un punto $ P_0(x_0, y_0, z_0) $
  - Vector normal $ \vec{n} = (a, b, c) $

### 1.1. Ecuación vectorial
- $\vec{n} \cdot (\vec{r} - \vec{r_0}) = 0$
donde $ \vec{r} = (x, y, z) $, $ \vec{r_0} = (x_0, y_0, z_0) $

### 1.2. Ecuación general (cartesiana)
- $
a(x - x_0) + b(y - y_0) + c(z - z_0) = 0
$
$
ax + by + cz + d = 0
$
donde $ d = -ax_0 - by_0 - cz_0 $

### 1.3. Ecuación segmentaria
- $
\frac{x}{p} + \frac{y}{q} + \frac{z}{r} = 1
$
- $ p, q, r $: intersecciones con ejes
- Ejemplo: $ 2x + 3y + 4z = 12 $
  $
  \frac{x}{6} + \frac{y}{4} + \frac{z}{3} = 1
  $

## 2. Casos particulares
### 2.1. Planos paralelos a ejes
- Paralelo a XY: $ z = k $
- Paralelo a XZ: $ y = k $
- Paralelo a YZ: $ x = k $

### 2.2. Planos que pasan por el origen
- $
ax + by + cz = 0
$
(d = 0)

### 2.3. Planos paralelos
- $
ax + by + cz + d_1 = 0
$
$
ax + by + cz + d_2 = 0
$
Mismo vector normal

## 3. Determinación de un plano
### 3.1. Por tres puntos no colineales
- Dados $ P_1, P_2, P_3 $:
$
\vec{n} = (\vec{P_2} - \vec{P_1}) \times (\vec{P_3} - \vec{P_1})
$

### 3.2. Por una recta y un punto exterior
- Vector normal perpendicular a la recta

### 3.3. Por dos rectas paralelas
- Vector normal perpendicular a ambas

### 3.4. Por dos rectas que se cortan
- Vector normal perpendicular a ambas direcciones

## 4. Posiciones relativas
### 4.1. Paralelismo entre planos
- $
\Pi_1: a_1x + b_1y + c_1z + d_1 = 0
$
$
\Pi_2: a_2x + b_2y + c_2z + d_2 = 0
$
Paralelos si:
$
\frac{a_1}{a_2} = \frac{b_1}{b_2} = \frac{c_1}{c_2} \neq \frac{d_1}{d_2}
$

### 4.2. Perpendicularidad
- Perpendiculares si:
$
a_1a_2 + b_1b_2 + c_1c_2 = 0
$

### 4.3. Ángulo entre planos
- $
\cos \theta = \frac{|a_1a_2 + b_1b_2 + c_1c_2|}{\sqrt{a_1^2 + b_1^2 + c_1^2} \sqrt{a_2^2 + b_2^2 + c_2^2}}
$

## 5. Distancias
### 5.1. De un punto a un plano
- Punto $ Q(x_1, y_1, z_1) $, plano $ ax + by + cz + d = 0 $:
$
d = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2 + b^2 + c^2}}
$

### 5.2. Entre planos paralelos
- $
d = \frac{|d_2 - d_1|}{\sqrt{a^2 + b^2 + c^2}}
$

# Soluciones de los ejercicios propuestos

## Nivel básico

### Ejercicio 1
- Hallar ecuación del plano perpendicular a $ \vec{n} = (1, 2, -3) $ que pasa por $ (2, -1, 4) $

- **Solución:**
 $
(x - 2) + 2(y + 1) - 3(z - 4) = 0
$
$
x - 2 + 2y + 2 - 3z + 12 = 0
$
$
x + 2y - 3z + 12 = 0
$

### Ejercicio 2
- Encontrar ecuación segmentaria de $ 3x - 4y + 6z = 12 $

- **Solución:**
  Dividimos toda la ecuación entre 12:
  $
  \frac{3x}{12} - \frac{4y}{12} + \frac{6z}{12} = 1
  $
  $
  \frac{x}{4} - \frac{y}{3} + \frac{z}{2} = 1
  $
  La ecuación segmentaria es:
  $
  \frac{x}{4} + \frac{y}{-3} + \frac{z}{2} = 1
  $
  Intersecciones con ejes: $ (4, 0, 0) $, $ (0, -3, 0) $, $ (0, 0, 2) $

### Ejercicio 3
- Calcular distancia del punto $ (1, 1, 1) $ al plano $ 2x - y + 2z + 3 = 0 $

- **Solución:**
Fórmula: $ d = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2 + b^2 + c^2}} $
$
d = \frac{|2(1) - 1(1) + 2(1) + 3|}{\sqrt{2^2 + (-1)^2 + 2^2}}
$
$
d = \frac{|2 - 1 + 2 + 3|}{\sqrt{4 + 1 + 4}}
$
$
d = \frac{|6|}{\sqrt{9}} = \frac{6}{3} = 2
$

## Nivel intermedio

### Ejercicio 4
- Hallar plano que contiene a los puntos $ A(1, 1, 0) $, $ B(2, 0, 1) $, $ C(0, 2, 1) $

- **Solución:**
Vectores:
$
\vec{AB} = (2-1, 0-1, 1-0) = (1, -1, 1)
$
$
\vec{AC} = (0-1, 2-1, 1-0) = (-1, 1, 1)
$
Vector normal:
$
\vec{n} = \vec{AB} \times \vec{AC} = 
\begin{vmatrix}
\vec{i} & \vec{j} & \vec{k} \\
1 & -1 & 1 \\
-1 & 1 & 1
\end{vmatrix}
$
$
\vec{n} = \vec{i}(-1\cdot 1 - 1\cdot 1) - \vec{j}(1\cdot 1 - 1\cdot (-1)) + \vec{k}(1\cdot 1 - (-1)\cdot (-1))
$
$
\vec{n} = \vec{i}(-1 - 1) - \vec{j}(1 + 1) + \vec{k}(1 - 1)
$
$
\vec{n} = (-2, -2, 0) = -2(1, 1, 0)
$
Usamos $ \vec{n} = (1, 1, 0) $ (simplificado)
Ecuación del plano con punto A(1, 1, 0):
$
1(x - 1) + 1(y - 1) + 0(z - 0) = 0
$
$
x - 1 + y - 1 = 0
$
$
x + y - 2 = 0
$

### Ejercicio 5
- Determinar si planos $ 2x - 3y + z = 5 $ y $ 4x - 6y + 2z = 7 $ son paralelos

- **Solución:**
Comparamos coeficientes:
Plano 1: $ a_1 = 2, b_1 = -3, c_1 = 1 $
Plano 2: $ a_2 = 4, b_2 = -6, c_2 = 2 $

  Verificamos proporcionalidad:
  $
  \frac{a_1}{a_2} = \frac{2}{4} = \frac{1}{2}
  $
  $
  \frac{b_1}{b_2} = \frac{-3}{-6} = \frac{1}{2}
  $
  $
  \frac{c_1}{c_2} = \frac{1}{2} = \frac{1}{2}
  $
  Los coeficientes son proporcionales, pero verifiquemos los términos independientes:
  $
  \frac{d_1}{d_2} = \frac{-5}{-7} = \frac{5}{7} \neq \frac{1}{2}
  $
**Sí son planos paralelos** (no coincidentes).

### Ejercicio 6
- Encontrar ángulo entre planos $ x + y + z = 1 $ y $ x - y + z = 2 $

- **Solución:**
Vectores normales: $ \vec{n_1} = (1, 1, 1) $, $ \vec{n_2} = (1, -1, 1) $
Fórmula: $ \cos \theta = \frac{|a_1a_2 + b_1b_2 + c_1c_2|}{\sqrt{a_1^2 + b_1^2 + c_1^2} \sqrt{a_2^2 + b_2^2 + c_2^2}} $
$
\cos \theta = \frac{|1\cdot 1 + 1\cdot (-1) + 1\cdot 1|}{\sqrt{1^2 + 1^2 + 1^2} \sqrt{1^2 + (-1)^2 + 1^2}}
$
$
\cos \theta = \frac{|1 - 1 + 1|}{\sqrt{3} \sqrt{3}} = \frac{1}{3}
$
$
\theta = \arccos\left(\frac{1}{3}\right) \approx 70.53^\circ
$

## Nivel avanzado

### Ejercicio 7
- Hallar ecuación del plano que contiene a la recta $ \frac{x-1}{2} = \frac{y}{-1} = \frac{z+2}{3} $ y es paralelo a la recta $ x = 2t, y = 1-t, z = 3+2t $

- **Solución:**
  1. **Recta contenida en el plano:** vector director $ \vec{v_1} = (2, -1, 3) $, punto $ P_1(1, 0, -2) $
  2. **Recta paralela al plano:** vector director $ \vec{v_2} = (2, -1, 2) $
  3. **Vector normal del plano:** perpendicular a ambos vectores directores
  $
  \vec{n} = \vec{v_1} \times \vec{v_2} = 
  \begin{vmatrix}
  \vec{i} & \vec{j} & \vec{k} \\
  2 & -1 & 3 \\
  2 & -1 & 2
  \end{vmatrix}
  $
  $
  \vec{n} = \vec{i}((-1)\cdot 2 - 3\cdot (-1)) - \vec{j}(2\cdot 2 - 3\cdot 2) + \vec{k}(2\cdot (-1) - (-1)\cdot 2)
  $
  $
  \vec{n} = \vec{i}(-2 + 3) - \vec{j}(4 - 6) + \vec{k}(-2 + 2)
  $
  $
  \vec{n} = (1, 2, 0)
  $
  4. Ecuación del plano con punto $ P_1(1, 0, -2) $:
  $
  1(x - 1) + 2(y - 0) + 0(z + 2) = 0
  $
  $
  x - 1 + 2y = 0
  $
  $
  x + 2y - 1 = 0
  $

### Ejercicio 8
- Calcular distancia entre planos paralelos $ 2x - y + 2z = 4 $ y $ 4x - 2y + 4z = 10 $

  **Solución:**
  Primero verificamos que sean paralelos. Simplificamos el segundo plano dividiendo entre 2:
  $
  \frac{4x - 2y + 4z = 10}{2} \Rightarrow 2x - y + 2z = 5
  $
  Ahora tenemos:
  Plano 1: $ 2x - y + 2z - 4 = 0 $ (d₁ = -4)
  Plano 2: $ 2x - y + 2z - 5 = 0 $ (d₂ = -5)

  Fórmula de distancia entre planos paralelos:
  $
  d = \frac{|d_2 - d_1|}{\sqrt{a^2 + b^2 + c^2}}
  $
  $
  d = \frac{|-5 - (-4)|}{\sqrt{2^2 + (-1)^2 + 2^2}}
  $
  $
  d = \frac{|-1|}{\sqrt{4 + 1 + 4}} = \frac{1}{\sqrt{9}} = \frac{1}{3}
  $

### Ejercicio 9
- Hallar plano que pasa por la intersección de $ x + 2y - z = 3 $ y $ 2x - y + z = 1 $, y es perpendicular a $ 3x - y + 2z = 4 $

- **Solución:**
  1. **Haz de planos por la recta intersección:**
  $
  (x + 2y - z - 3) + \lambda(2x - y + z - 1) = 0
  $
  $
  (1 + 2\lambda)x + (2 - \lambda)y + (-1 + \lambda)z + (-3 - \lambda) = 0
  $

  2. **Condición de perpendicularidad** con $ 3x - y + 2z = 4 $:
  $
  a_1a_2 + b_1b_2 + c_1c_2 = 0
  $
  $
  (1 + 2\lambda)\cdot 3 + (2 - \lambda)\cdot (-1) + (-1 + \lambda)\cdot 2 = 0
  $

  3. **Resolvemos para λ:**
  $
  3 + 6\lambda - 2 + \lambda - 2 + 2\lambda = 0
  $
  $
  (3 - 2 - 2) + (6\lambda + \lambda + 2\lambda) = 0
  $
  $
  -1 + 9\lambda = 0
  $
  $
  \lambda = \frac{1}{9}
  $

  4. **Sustituimos en la ecuación del haz:**
  $
  \left(1 + 2\cdot\frac{1}{9}\right)x + \left(2 - \frac{1}{9}\right)y + \left(-1 + \frac{1}{9}\right)z + \left(-3 - \frac{1}{9}\right) = 0
  $
  $
  \left(\frac{9}{9} + \frac{2}{9}\right)x + \left(\frac{18}{9} - \frac{1}{9}\right)y + \left(-\frac{9}{9} + \frac{1}{9}\right)z + \left(-\frac{27}{9} - \frac{1}{9}\right) = 0
  $
  $
  \frac{11}{9}x + \frac{17}{9}y - \frac{8}{9}z - \frac{28}{9} = 0
  $

  5. **Multiplicamos por 9:**
  $
  11x + 17y - 8z - 28 = 0
  $
