## Pregunta 1
- 3.2_3.3 Calcula la transformada de Laplace de $ f(t) =  e^{-3t} $.
- **Opciones:**
  - a) $ \frac{1}{(s+3)^2} $
  - b) $ \frac{1}{s^2} $
  - c) $ \frac{1}{s+3} $
  - d) $ \frac{s}{(s+3)^2} $
- **Solución:** La opción correcta es **c**. Usando la propiedad $ \mathcal{L}\{t e^{at}\} = \frac{1}{(s-a)^2} $, con $ a = -3 $, se obtiene $ \frac{1}{(s+3)^2} $.

## Pregunta 2
- 3.2_3.3 Calcula la transformada de Laplace de $ f(t) = \sin(4t) $.
- **Opciones:**
  - a) $ \frac{4}{s^2 + 16} $
  - b) $ \frac{s}{s^2 + 16} $
  - c) $ \frac{4}{s^2 + 4} $
  - d) $ \frac{s}{s^2 + 4} $
- **Solución:** La opción correcta es **a**. La transformada de $ \sin(kt) $ es $ \frac{k}{s^2 + k^2} $, con $ k=4 $, resulta $ \frac{4}{s^2 + 16} $.

## Pregunta 3
- 3.2_3.3 Calcula la transformada inversa de Laplace de $ F(s) = \frac{3}{s^2 + 9} $.
- **Opciones:**
  - a) $ \sin(3t) $
  - b) $ \cos(3t) $
  - c) $ \frac{1}{3} \sin(3t) $
  - d) $ 3 \sin(3t) $
- **Solución:** La opción correcta es **a**. Sabemos que $ \mathcal{L}^{-1}\left\{ \frac{k}{s^2 + k^2} \right\} = \sin(kt) $, aquí $ k=3 $, pero $ F(s) = \frac{3}{s^2 + 9} = \frac{3}{s^2 + 3^2} $, así que $ \mathcal{L}^{-1} = \sin(3t) $. Opción a es correcta.

## Pregunta 5
- 3.2_3.3 Calcula la transformada de Laplace de $ f(t) = \cos(2t) $.
- **Opciones:**
  - a) $ \frac{s}{s^2 - 4} $
  - b) $ \frac{s}{s^2 + 4} $
  - c) $ \frac{2}{s^2 - 4} $
  - d) $ \frac{2}{s^2 + 4} $
- **Solución:** La opción correcta es **b**. Ya que $ \cosh(kt) = \frac{e^{kt} + e^{-kt}}{2} $, su transformada es $ \frac{s}{s^2 - k^2} $, con $ k=2 $, resulta $ \frac{s}{s^2 - 4} $.

## Pregunta 6
- ¿Cuál es la definición correcta de la transformada de Laplace de una función $ f(t) $?
- **Opciones:**
  - a) $ \int_{-\infty}^{\infty} f(t) e^{-st} \, dt $
  - b) $ \int_{0}^{\infty} f(t) e^{st} \, dt $
  - c) $ \int_{0}^{\infty} f(t) e^{-st} \, dt $
  - d) $ \int_{-\infty}^{0} f(t) e^{-st} \, dt $
- **Solución:** La opción correcta es **c**. La transformada de Laplace se define como $ \mathcal{L}\{f(t)\} = \int_{0}^{\infty} f(t) e^{-st} \, dt $.

## Pregunta 7
- 3.2_3.3 Calcula la transformada de Laplace de $ f(t) = t^2 + 3t + 2 $.
- **Opciones:**
  - a) $ \frac{2}{s^3} + \frac{3}{s^2} + \frac{2}{s} $
  - b) $ \frac{2}{s^3} + \frac{3}{s^2} + \frac{1}{s} $
  - c) $ \frac{1}{s^3} + \frac{3}{s^2} + \frac{2}{s} $
  - d) $ \frac{2}{s^3} + \frac{1}{s^2} + \frac{2}{s} $
- **Solución:** La opción correcta es **a**. Usando $ \mathcal{L}\{t^n\} = \frac{n!}{s^{n+1}} $, para $ t^2 $: $ \frac{2!}{s^3} = \frac{2}{s^3} $, para $ 3t $: $ \frac{3}{s^2} $, para $ 2 $: $ \frac{2}{s} $.

## Pregunta 8
- 3.2_3.3 ¿Cuál es la propiedad de linealidad de la transformada de Laplace?
- **Opciones:**
  - a) $ \mathcal{L}\{af(t) + bg(t)\} = a\mathcal{L}\{f(t)\} + b\mathcal{L}\{g(t)\} $
  - b) $ \mathcal{L}\{f(t)g(t)\} = \mathcal{L}\{f(t)\} \cdot \mathcal{L}\{g(t)\} $
  - c) $ \mathcal{L}\{f(t) + g(t)\} = \mathcal{L}\{f(t)\} + \mathcal{L}\{g(t)\} $
  - d) $ \mathcal{L}\{af(t)\} = a\mathcal{L}\{f(t)\} $
- **Solución:** La opción correcta es **a**. La transformada de Laplace es lineal: $ \mathcal{L}\{af(t) + bg(t)\} = a\mathcal{L}\{f(t)\} + b\mathcal{L}\{g(t)\} $.
