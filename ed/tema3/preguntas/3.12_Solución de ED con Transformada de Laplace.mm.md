# Preguntas sobre Solución de ED con Transformada de Laplace

## Pregunta 1
- 3.12 Resuelve la ED de primer orden usando Laplace:  
  $$\frac{dy}{dt} + 2y = 0, \quad y(0)=1$$  
**Opciones:**  
- a) $ y(t) = e^{-2t} $  
- b) $ y(t) = e^{2t} $  
- c) $ y(t) = e^{-t} $  
- d) $ y(t) = e^{t} $  
**Solución:**  
Aplicando Laplace:  
$ sY(s) - 1 + 2Y(s) = 0 $  
$ (s+2)Y(s) = 1 $  
$ Y(s) = \frac{1}{s+2} $  
Antitransformada: $ y(t) = e^{-2t} $  
**Respuesta correcta:** a)

## Pregunta 2
- 3.12 Resuelve la ED de segundo orden usando Laplace:  
  $$y'' + 4y = 0, \quad y(0)=0, \quad y'(0)=1$$  
**Opciones:**  
- a) $ y(t) = \frac{1}{2}\sin(2t) $  
- b) $ y(t) = \frac{1}{2}\cos(2t) $  
- c) $ y(t) = \sin(2t) $  
- d) $ y(t) = \cos(2t) $  
**Solución:**  
Aplicando Laplace:  
$ s^2Y(s) - 1 + 4Y(s) = 0 $  
$ (s^2+4)Y(s) = 1 $  
$ Y(s) = \frac{1}{s^2+4} $  
Antitransformada: $ y(t) = \frac{1}{2}\sin(2t) $  
**Respuesta correcta:** a)

## Pregunta 3
- 3.12 Resuelve la ED con forzamiento:  
  $$\frac{dy}{dt} + 3y = e^{-t}, \quad y(0)=0$$  
**Opciones:**  
- a) $ y(t) = \frac{1}{2}(e^{-t} - e^{-3t}) $  
- b) $ y(t) = \frac{1}{2}(e^{-t} + e^{-3t}) $  
- c) $ y(t) = \frac{1}{2}(e^{t} - e^{-3t}) $  
- d) $ y(t) = \frac{1}{2}(e^{-t} - e^{3t}) $  
**Solución:**  
Aplicando Laplace:  
$ sY(s) + 3Y(s) = \frac{1}{s+1} $  
$ Y(s) = \frac{1}{(s+1)(s+3)} = \frac{1}{2}\left(\frac{1}{s+1} - \frac{1}{s+3}\right) $  
Antitransformada: $ y(t) = \frac{1}{2}(e^{-t} - e^{-3t}) $  
**Respuesta correcta:** a)

## Pregunta 4
- 3.12 Resuelve la ED de segundo orden con forzamiento:  
  $$y'' + 2y' + y = e^{-t}, \quad y(0)=0, \quad y'(0)=0$$  
**Opciones:**  
- a) $ y(t) = \frac{1}{2}t^2 e^{-t} $  
- b) $ y(t) = \frac{1}{2}t e^{-t} $  
- c) $ y(t) = t^2 e^{-t} $  
- d) $ y(t) = t e^{-t} $  
**Solución:**  
Aplicando Laplace:  
$ (s+1)^2 Y(s) = \frac{1}{s+1} $  
$ Y(s) = \frac{1}{(s+1)^3} $  
Antitransformada: $ y(t) = \frac{1}{2}t^2 e^{-t} $  
**Respuesta correcta:** a)

## Pregunta 5
- 3.12 Resuelve el sistema de ED:  
  $$
  \begin{cases}
  x' = -y \\
  y' = x
  \end{cases}
  \quad \text{con} \quad x(0)=1, \quad y(0)=0
$$  
**Opciones:**  
- a) $ x(t) = \cos(t), \quad y(t) = \sin(t) $  
- b) $ x(t) = \sin(t), \quad y(t) = \cos(t) $  
- c) $ x(t) = \cos(t), \quad y(t) = -\sin(t) $  
- d) $ x(t) = \sin(t), \quad y(t) = -\cos(t) $  
**Solución:**  
Aplicando Laplace:  
$ sX(s) - 1 = -Y(s) $  
$ sY(s) = X(s) $  
Sustituyendo: $ s^2 Y(s) - 1 = -Y(s) $  
$ (s^2+1)Y(s) = 1 $  
$ Y(s) = \frac{1}{s^2+1} \Rightarrow y(t) = \sin(t) $  
$ X(s) = \frac{s}{s^2+1} \Rightarrow x(t) = \cos(t) $  
**Respuesta correcta:** a)

## Pregunta 6
- 3.12 Resuelve el sistema de ED:  
  $$
  \begin{cases}
  x' = 2x + y \\
  y' = x + 2y
  \end{cases}
  \quad \text{con} \quad x(0)=1, \quad y(0)=0
  $$  
**Opciones:**  
- a) $ x(t) = \frac{1}{2}(e^{3t} + e^{t}), \quad y(t) = \frac{1}{2}(e^{3t} - e^{t}) $  
- b) $ x(t) = \frac{1}{2}(e^{3t} - e^{t}), \quad y(t) = \frac{1}{2}(e^{3t} + e^{t}) $  
- c) $ x(t) = \frac{1}{2}(e^{3t} + e^{t}), \quad y(t) = \frac{1}{2}(e^{3t} + e^{t}) $  
- d) $ x(t) = \frac{1}{2}(e^{3t} - e^{t}), \quad y(t) = \frac{1}{2}(e^{3t} - e^{t}) $  
**Solución:**  
Aplicando Laplace:  
$ (s-2)X(s) - Y(s) = 1 $  
$ -X(s) + (s-2)Y(s) = 0 $  
Resolviendo:  
$ Y(s) = \frac{1}{(s-1)(s-3)} = \frac{1}{2}\left(\frac{1}{s-3} - \frac{1}{s-1}\right) $  
$ y(t) = \frac{1}{2}(e^{3t} - e^{t}) $  
$ X(s) = \frac{1}{2}\left(\frac{1}{s-3} + \frac{1}{s-1}\right) $  
$ x(t) = \frac{1}{2}(e^{3t} + e^{t}) $  
**Respuesta correcta:** a)

## Pregunta 7
- 3.12 Resuelve la ED de segundo orden:  
  $$y'' - 3y' + 2y = 0, \quad y(0)=1, \quad y'(0)=0$$  
**Opciones:**  
- a) $ y(t) = 2e^{t} - e^{2t} $  
- b) $ y(t) = 2e^{2t} - e^{t} $  
- c) $ y(t) = 2e^{t} + e^{2t} $  
- d) $ y(t) = 2e^{2t} + e^{t} $  
**Solución:**  
Aplicando Laplace:  
$ (s^2 - 3s + 2)Y(s) = s - 3 $  
$ Y(s) = \frac{s-3}{(s-1)(s-2)} = \frac{2}{s-1} - \frac{1}{s-2} $  
Antitransformada: $ y(t) = 2e^{t} - e^{2t} $  
**Respuesta correcta:** a)

## Pregunta 8
- 3.12 Resuelve la ED de primer orden:  
  $$\frac{dy}{dt} + 2y = 3, \quad y(0)=0$$  
**Opciones:**  
- a) $ y(t) = \frac{3}{2}(1 - e^{-2t}) $  
- b) $ y(t) = \frac{3}{2}(1 + e^{-2t}) $  
- c) $ y(t) = \frac{3}{2}(1 - e^{2t}) $  
- d) $ y(t) = \frac{3}{2}(1 + e^{2t}) $  
**Solución:**  
Aplicando Laplace:  
$ sY(s) + 2Y(s) = \frac{3}{s} $  
$ Y(s)(s+2) = \frac{3}{s} $  
$ Y(s) = \frac{3}{s(s+2)} = \frac{3}{2}\left(\frac{1}{s} - \frac{1}{s+2}\right) $  
Antitransformada: $ y(t) = \frac{3}{2}(1 - e^{-2t}) $  
**Respuesta correcta:** a)
