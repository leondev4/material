### **Pregunta 1:**
- Hallar los puntos críticos de $ f(x) = x^3 - 4x^2 - 7x + 10$.

  **A)** $x = -1, 2, 5$
  **B)** $x = -1, 1, 7$
  **C)** $x = -2, 1, 5$
  **D)** $x = -1, 2, 4$

  **Respuesta correcta: A**  
  $
  f'(x) = 3x^2 - 8x - 7 = 0 \Rightarrow x = -1, 2, 5
  $

### **Pregunta 2:**
- Hallar el área bajo la curva de $f(x) = x e^x$ en $1 \leq x \leq 3$.

  **A)** $2e^3$
  **B)** $2e^3 - e$
  **C)** $e^3 - e$
  **D)** $3e^3 - 2e$

  **Respuesta correcta: B**  
  $
  \int_{1}^{3} x e^x \, dx = \left[ (x-1)e^x \right]_{1}^{3} = 2e^3 - e
  $

### **Pregunta 3:**
- Hallar la derivada de $f(x) = x^2 \cos(2x)$.

   **A)** $2x \cos(2x) - 2x^2 \sin(2x)$
   **B)** $2x \cos(2x) + x^2 \sin(2x)$
   **C)** $2x \cos(2x) - x^2 \sin(2x)$
   **D)** $x \cos(2x) - 2x^2 \sin(2x)$

  **Respuesta correcta: A**  
  $
  f'(x) = 2x \cos(2x) - 2x^2 \sin(2x)
  $

### **Pregunta 4:**
- Hallar la derivada parcial $\frac{\partial f}{\partial y}$ de $f(x, y) = x y \cos(xy) - e^{xy}$.

  **A)** $x \cos(xy) - x^2 y \sin(xy) - x e^{xy}$
  **B)** $x \cos(xy) - x y \sin(xy) - e^{xy}$
  **C)** $x \cos(xy) - x^2 y \sin(xy) - e^{xy}$
  **D)** $x \cos(xy) - x y \sin(xy) - x e^{xy}$

  **Respuesta correcta: A**  
  $
  \frac{\partial f}{\partial y} = x \cos(xy) - x^2 y \sin(xy) - x e^{xy}
  $

### **Pregunta 5:**
- Calcular la integral de $f(x) = \int x \cos(x^2) \, dx$.

  **A)** $\frac{1}{2} \sin(x^2) + C$
  **B)** $\frac{1}{2} \cos(x^2) + C$
  **C)** $\sin(x^2) + C$
  **D)** $\cos(x^2) + C$

  **Respuesta correcta: A**  
  $
  \int x \cos(x^2) \, dx = \frac{1}{2} \sin(x^2) + C
  $

### **Pregunta 7:**
- Calcular el valor de $C$ de la ecuación $5 = C e^{2x}$ cuando $x = 0$.

  **A)** $C = 5$
  **B)** $C = 2$
  **C)** $C = 0$
  **D)** $C = 1$

  **Respuesta correcta: A**  
  $
  5 = C e^{0} \Rightarrow C = 5
